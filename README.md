# PDF Converter

在线 PDF 转换工具 — 将 PDF 转换为 Word 或 PowerPoint。

**在线访问：https://about-winter.github.io/pdf-converter**

## 功能特点

- 拖拽或点击上传 PDF 文件
- 导出为 Word (.docx) 或 PowerPoint (.pptx)
- 完全在浏览器本地处理，文件不上传服务器
- 转换后自动下载

## 技术栈

- pdf.js — PDF 文本提取
- docx — Word 文档生成
- pptxgenjs — PowerPoint 生成

---
基于 pdf.js、docx 和 pptxgenjs 构建。
